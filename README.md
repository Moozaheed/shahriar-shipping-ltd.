## 🚀 Deployment

The site is deployed on **Netlify** via CLI.

**Live URL:** https://shahriar-shipping-ltd.netlify.app

### Run Locally
```bash
# Clone the repo
git clone https://github.com/Moozaheed/shahriar-shipping-ltd..git
cd shahriar-shipping-ltd.

# Serve locally (Python)
python3 -m http.server 8080
# Then open http://localhost:8080
```

### Redeploy to Netlify
```bash
netlify deploy --prod --dir .
```
